# Smart Password Gen

##  Nom et description du projet
**Smart Password Gen** est un générateur de mots de passe simple, sécurisé et interactif.  
L’outil permet de créer des mots de passe forts en fonction de critères choisis : longueur, types de caractères (majuscules, minuscules, chiffres, symboles).

---

##  Technologies utilisées
- **HTML5**
- **CSS3** (design moderne + animations légères)
- **JavaScript (ES6)**

---

##  Fonctionnalités principales
-  Génération automatique de mots de passe robustes  
-  Choix de la longueur et des catégories de caractères  
-  Affichage des détails : longueur, niveau de sécurité, estimation du temps  
-  Historique des mots de passe générés  
-  Copie facile vers le presse‑papier  
-  Interface responsive (mobile & desktop)

##  Nouveautés explorées
Durant ce projet, j’ai appris et exploré :
-  L’utilisation de **CSS Grid et Flexbox** pour structurer la mise en page
-  Les **animations CSS simples** pour une meilleure UX
-  La manipulation du DOM avec **JavaScript** pour générer le mot de passe et mettre à jour l’interface
-  L’API **navigator.clipboard** pour copier les résultats

---

##  Difficultés rencontrées
-  Déterminer une logique simple mais significative pour évaluer la **force du mot de passe**
-  Gérer l’historique des mots de passe avec ajout dynamique
-  Rendre l’interface fluide et responsive sur petits écrans

---

##  Solutions apportées
-  Implémentation d’un score simple basé sur longueur + types de caractères
-  Utilisation de `prepend()` pour garder l’historique bien organisé
-  Media queries CSS pour adapter le layout en responsive
-  Recherche dans la documentation MDN pour maîtriser les APIs nécessaires
